# 距離センサー LidarLiteV3

<hr>

LidarLiteV3はJetson TX2で正常に値を取得することは出来ません。<br>
他ライブラリの読み込みのために入れるだけにします。<br>

# pipインストール
```
mkdir -p ~/notebooks/github
cd ~/notebooks/github
git clone https://github.com/FaBoPlatform/LidarLiteV3
cd LidarLiteV3
sudo pip install .
```
